Game of the week

Plantstead by Nadia Nova

A quick project:

  • Start a new empty project
  • Add a terrain (Create -> 3d -> Terrain), draw some mountains
  • Import the standard characters (Assets -> Import Package -> Characters)
  • Add a First Person Controller (note the instructions file telling you to delete the main camera)
  • Demo running around
  • Add textures to the terrain
  • opengameart.org - check licences
  • Set up a skybox
  • (how to Google) -> https://docs.unity3d.com/Manual/HOWTO-UseSkybox.html
  • Skybox = Material, so add a Material to the Project and change the shader to Skybox-6-sided
  • Use your skybox: Window -> Lighting
  • Add a 3d model (car?) Try adding physics (a mesh collider, then a rigidbody. Meshcolliders don't always work, try a box collider.)
  • Attach a spotlight to the car (headlights)
  • Revise anything from Roll-a-ball
  • Let’s look back at Plantstead’s itch.io page. Note the list of assets used? Let’s check them out.
