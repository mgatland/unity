Finding out how to do stuff

There are lots of online discussions, and great documentation, so searching online is the best way to find answers. You can explore the Unity Manual for general Unity info, and the Unity Scripting API for information on scripting (writing code).

If you are searching online using a search engine like google, put 'Unity' or 'Unity3D' in your query. For example, "Unity3d look at another gameobject."

Unity Collaborate

Unity Collaborate is a tool designed to help teams work together on a Unity project. It is free for team of up to 4 people with projects of up to 1 GB (which is quite small.)

Unfortunately Unity Collaborate is not working properly at AUT. You can still use it, but you need to close and reopen Unity every time you want to see the current collab state (every time you download a teammate's changes or publish your own changes. This can be annoying.

If you want to try using Unity Collaborate anyway, click through to the official Unity Collaborate documentation to learn how to use it.

Other collaboration ideas

If you do not to use Unity Collaborate, you need to think about how you will get everyone's work together into the same project. Some tips:

  • Make it easy for any team member to add their work to the project. For example, put the project in a Dropbox folder that everyone can edit. Dropbox automatically copies changed files to everyone's computer.
  • If you use Dropbox, you must avoid having two people working on the project at the same time.
  • Make regular backups!
